## User Stories
- **Workflow Theme**
1. As a student, I want to be reminded of what to look for while writing a reference note, so that I stay mindful of the distinction between main and secondary ideas.
2. As a student I need to keep track of where my information comes from, so that I can go back to the source if I need extra clarifications and to have it ready as a reference for my projects. 
3. As a student I want to be asked whether I understood a reference note after saving it, so that I can immediately link it to an own words note, a question note, or defer the decision for later.
4. As a student I want to be able to defer the understanding decision on a reference note, so that I can return to it later without pressure when I'm not ready to evaluate my understanding.
5. As a student I want to be able to link my own words notes and questions notes to the reference notes they stem from, so that I can compare the two.
6. As a student I want to be notified of the unanswered questions, so that I prioritize them when revising.
7. As a student I want to be able to check questions as answered after linking them to own words notes, so that I make a conscious decision that the respective idea is understood.
8. As a student I want to be able to differentiate between linked and unlinked reference notes, so that I make the unlinked ones a priority.
9. As a student I want to be able to turn my question notes into own words notes when I find the answers, so that I have a clear picture of what I understand and what I don't.
10. As a student I want to be able to create an own words note, optionally linked to a reference note, so that I can either process a source or capture my own thinking independently.
11. As a student I want to be able to create a question note, optionally linked to a reference note, so that I can track gaps in my understanding whether they arise from a source or independently.<br>
- **UI/UX Theme**
12. As a student I want to be able to see a list of all the reference notes, own words notes and question notes related to a particular course, so that I have an ensemble view of what's done and what's left.
13. As a student I want a quick capture option for own words and question notes directly from the dashboard, so that I don't lose a thought while navigating the app.
14. As a student I want to be able to quickly access my recent notes, so that I pick up where I left off.
15. As a new user I want a an intuitive design, so that I can start taking notes quickly, without frustration.
16. As a new user I want to be given a walkthrough on the home page so that i know what the app is about and how to start using it.
17. As a user I need a responsive design, so that I can use the app on a multitude of devices.
18. As a student with learning difficulties I need an accessible design, so that I can use the app without problems.<br>
- **CRUD Functionality Theme**<br>
- **Notes**
19. As a student I want to be able to modify any note, so that I keep my notes up to date with my understanding.
20. As a student I want to be able to delete any of my notes with a confirmation step, so that I can declutter my notes without accidentally losing them.
21. As a student I want to be able to search for a specific note, so that I can easily find it when I need it.<br>
- **Courses**
22. As a student I want to be able to create a course, so that I can organise my notes by subject.
23. As a student I want to be able to rename a course, so that I can keep it accurate if the name changes.
24. As a student I want to be asked to confirm before deleting a course, so that I don't accidentally lose all the notes within it.
25. As a student I want to be able to delete a course when I no longer need it, so that my dashboard stays uncluttered.<br>
- **Units**
26. As a student I want to be able to create a unit within a course, so that I can organise my notes by topic.
27. As a student I want to be able to rename a unit, so that I can keep it aligned with my course structure.
28. As a student I want to be asked to confirm before deleting a unit, so that I don't accidentally lose all the notes within it.
29. As a student I want to be able to delete a unit when I no longer need it, so that I can keep my course structure tidy.<br>
- **Tags**
30. As a student I want to be able to assign one or more tags to a note, so that I can connect related notes across different courses.
31. As a student I want to be prompted with a list of already used tags when tagging a note, so that I keep my tags consistent and avoid duplicates.
32. As a student I want to be able to remove a tag from a note, so that I can correct mistakes or update connections.
33. As a student I want to be able to see all notes associated with a tag in one view, so that I can explore connections between ideas across subjects.<br>
- **Authentication Theme**
34. As a new user I want to be able to create a new account, to start using the app.
35. As a user I want to be able to sign into my account, to be able to access my notes and create new ones.
36. As a user I want to be able to reset my password, so as not to lose access to my account when I forget it.
37. As a user I want to stay logged in between sessions, so that I don't have to sign in every time.
38. As a user I want to be prompted to reconsider whether I want to delete my account when I press delete account button, so that I don't accidentally lose my account and notes.
39. As a user I want to be able to delete my account when I consider I don't need it anymore, so that I can be in control of my information.
40. As a new user I want to verify my email address after registering, so that my account is secure and recoverable.
