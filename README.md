## User Stories
1. As a student I want to be prompted to evaluate the importance of a note before writing it, so that I stay mindful of what I'm learning and always distinguish between main and secondary ideas.
2. As a student I need to keep track of where my information comes from, so that I can go back to the source if I need extra clarifications and to have it ready as a reference for my projects. 
3. As a student I want to be prompted to check my understanding of the main ideas in the course, so that I clearly differentiate between what I understand and what I don't.
4. As a student I want to be able to link my own words notes and questions notes to the reference notes they stem form, so that I can compare the two.
5. As a student I want to be notified of the unanswered questions, so that I prioritize them when revising.
6. As a student I want to be able to turn my question notes into own words notes when I find the answers, so that I have a clear picture of what I understand and what I don't.
7. As a student I want to be able to see a list of all the reference notes, own words notes and question notes related to a particular course, so that I have an ensemble view of what's done and what's left.
8. As a student I want to be able to modify any note, so that I keep my notes up to date with my understanding.
9. As a student I want to be given an extra security question when trying to delete any of my notes, so that I don't accidentally delete notes.
10. As a student I want to be given the option to delete any of my notes, to declutter my dashboard when I think that is appropriate.
11. As a student I want to be able to search for a specific note, so that I can easily find it when I need it.
12. As a student I want to be able to be prompted a list of already used tags when I want to assign a new one, so that I avoid overlapping tags.
13. As a student I want to be able to filter search by tags, so that I can easily find related notes.
14. As a student I want to be able to create a note with my own words so that I prove to myself that I understand an idea.
15. As a student I want to be able to create a question note when I don't understand something, so that I keep track of gaps in my knowledge.
16. As a new user I want a an intuitive design, so that I can start taking notes quickly, without frustration.
17. As a new user I want to be given a walkthrough on the home page so that i know what the app is about and how to start using it.
18. As a user I need a responsive design, so that I can use the app on a multitude of devices.
19. As a student with learning difficulties I need an accessible design, so that I can use the app without problems.
20. As a new user I want to be able to create a new account, to start using the app.
21. As a user I want to be able to sign into my account, to be able to access my notes and create new ones.
22. As a user I want to be able to reset my password, so as not to lose access to my account when I forget it.
23. As a user I want to stay logged in between sessions, so that I don't have to sign in every time.
24. As a user I want to be prompted to reconsider whether I want to delete my account when I press delete account button, so that I don't accidentally lose my account and notes.
25. As a user I want to be able to delete my account when I consider I don't need it anymore, so that I can be in control of my information.
26. As a new user I want to verify my email address after registering, so that my account is secure and recoverable.
